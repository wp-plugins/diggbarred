Diggbarred stops all DiggBar requests to your site, instead displaying a message of your choosing.

== Description ==

This is a Wordpress plugin version of John Gruber's DiggBar blocking code, with some options for the user. With Diggbarred, the user can customize both the message displayed and the styling of the DIV element that contains the message.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `diggbarred` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it!
